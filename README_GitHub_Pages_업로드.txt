GitHub Pages 업로드용 정리본 v8.22-4_fixed

이 정리본에는 PWA 설치에 필요한 manifest.json, service-worker.js, .nojekyll, 아이콘 파일과 CAD 수치지형도 타일 폴더가 포함되어 있습니다.

GitHub Desktop 사용 방법:
1. 이 ZIP을 압축 해제합니다.
2. 압축 해제한 폴더 안의 내용물을 GitHub 저장소 루트에 모두 복사합니다.
   - 폴더 자체를 넣지 말고 안의 내용물을 넣어야 합니다.
3. GitHub Desktop에서 변경사항을 commit 후 push 합니다.
4. GitHub Pages 주소에서 새로고침합니다.

주의:
- manifest.json, service-worker.js, .nojekyll, cad_topo_tiles 폴더가 저장소 루트에 있어야 합니다.
- desktop.html만 올리면 PWA 설치와 CAD 수치지형도가 정상 작동하지 않습니다.
