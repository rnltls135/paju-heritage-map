GitHub Pages 업로드용 정리본

1. 이 ZIP을 압축 해제합니다.
2. 압축 해제한 폴더 안의 모든 파일과 폴더를 GitHub 저장소 루트에 올립니다.
3. 반드시 루트에 .nojekyll 파일이 있어야 합니다.
4. GitHub 저장소 Settings > Pages에서
   Source: Deploy from a branch
   Branch: main
   Folder: /root
   로 설정합니다.
5. desktop.html만 따로 올리면 CAD 수치지형도가 표시되지 않습니다.
   cad_topo_tiles 폴더까지 모두 같이 올려야 합니다.
